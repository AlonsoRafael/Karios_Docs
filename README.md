## Sobre

Plataforma web moderna para Kairós Docs - documentos prontos e profissionais para clínicas de saúde. Site institucional com foco em conversão, design responsivo e experiência do usuário otimizada.

## O que entrega

- ✨ Landing page moderna e responsiva
- 🎨 Design elegante com animações suaves
- 📱 Totalmente otimizado para mobile
- 🔄 Seção de produtos e packs
- 💬 FAQ com accordion interativo
- ⭐ Depoimentos de clientes
- 📊 Pricing com call-to-action estratégicos
- ⚡ Performance otimizada com Next.js 14

## Quick Start

```bash
git clone https://github.com/SEU_USUARIO/Karios_Docs.git
cd Karios_Docs
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.
